# Быстрый старт 3D Реконструкции

## 🚀 Самый быстрый способ запуска

### На Linux/Mac:
```bash
cd application
chmod +x run.sh
./run.sh
```

### На Windows:
```bash
cd application
run.bat
```

## 📖 Что произойдёт после запуска

1. ✅ Создастся виртуальное окружение Python (если его нет)
2. ✅ Установятся все зависимости
3. ✅ Запустится веб-сервер на http://localhost:8000
4. ✅ Интерфейс откроется в браузере

## 💡 Что дальше

1. **Загрузите видео** - кликните на зону загрузки или перетащите видеофайл
2. **Выберите алгоритм** - по умолчанию используется COLMAP 360 + OpenMVS
3. **Ждите результата** - смотрите прогресс в реальном времени
4. **Скачайте результат** - когда обработка завершится

## ⚙️ Требования

- **Python 3.8+**
- **4+ ГБ оперативной памяти**
- **20+ ГБ свободного места на диске** (зависит от размера видео)
- **OpenMVG, OpenMVS или другие зависимости** (устанавливаются отдельно)

## 🎯 Системные требования

### Для быстрой обработки:
- NVIDIA GPU (CUDA) - рекомендуется
- 16+ ГБ RAM
- SSD диск

### Минимально:
- 4 ГБ RAM
- HDD диск
- Обработка может занять часы

## 🔧 Кастомизация

### Добавить новый алгоритм обработки

1. Создайте новый pipeline в `backend/pipelines/`
2. Добавьте тип в `backend/models/database.py`
3. Обновите список в `web/src/components/TaskUploader.tsx`

### Изменить вид интерфейса

1. Отредактируйте стили в `web/src/components/*/css`
2. Запустите `npm run dev` в папке `web/`
3. Откройте http://localhost:5173

## 🐛 Решение проблем

### Ошибка "Port already in use"
```bash
# Linux/Mac
lsof -i :8000
kill -9 <PID>

# Windows
netstat -ano | findstr :8000
taskkill /PID <PID> /F
```

### Ошибка "ModuleNotFoundError"
```bash
# Переустановите зависимости
cd backend
pip install -r requirements.txt
```

### Низкая скорость обработки
- Убедитесь, что используется GPU (если доступна)
- Уменьшите разрешение видео перед загрузкой
- Выберите другой алгоритм
